#xFastTemplate2
This (snippet) module is a simple class for php/html/xhmtl/xml templates.

###Description
Module for Lepton-CMS
see: http://http://www.lepton-cms.org
and: http://http://forum.lepton-cms.org

####Require

- MySQL 5.x
- Lepton-CMS >= 1.3.1
